Redovisning av kursmoment i kursen PA1444

Kmom01

Hur känns det att komma igång med MySQL och dess klienter?
Det var lite svårt att få det att fungera i början med servern och terminalen och få allting att installeras korrekt men såfort man kom igång med själva mysql och förstod hur det fungerade så flöt det på ganska bra. Det gick att gissa sig fram till det mesta så det var roligt och kändes att man kunde med sina egna kunskaper inom dom andra programmeringskurserna jag har läst kunna lösa de olika problemen. Har du jobbat med databaser eller liknande tidigare?
Nej hade aldrig programmerat tidigare alls när jag började på skolan, men känner att jag har ganska lätt att förstå hur saker fungerar efter ett antal timmars programmerande och testande.
Hur gick det att jobba med SQL?
Bra, tyckte det var riktigt kul att fixa runt med det olika uppgifterna och använda terminalen för att få fram de olika listorna man skapat. Det var lite ovant i början men det kändes som om jag fick ordning på det i slutet.
Jämför SQL med andra sätt att programmera.
Vissa sätt att skriva olika funktioner kändes liknande från visual studio som jag har jobbat med tidigare, men det sättet att ha flera sql filer som sedan körs i tur och ordning för att skapa sitt program var lite nytt. Det var intressant hur det var uppbyggt men lite ovant.

Vilken är din TIL för detta kmom?
Min TIL i denna kmom är hur man navigerar i en terminal, hur man skriver basic sql koder och hur man editar en html hemsida. Så har lärt mig ganska mycket nu den här kmom känner jag!

Kmom02

Du har nu repeterat SQL ett par gånger, börjar det sätta sig, är det något du finner extra utmanande?
Jag tycker att jag börjar känna mig mer säkert på sql nu efter dessa uppgifterna, det är lättare att klura ut vad man ska göra och man kan jobba sig runt problemen lättare men jag tycker det är svårt att komma ihåg vad allt gör exakt, group by / order by te.x tycker jag är svårt att skilja åt och blandar alltid ihop dom. Men det kommer jag nog komma över efter lite träning.

Är detta den första bekantskap med JavaScript på serversidan och Node.js, hur känns det?
Ja det var första gången jag testade JavaScript och jag tyckte det var väldigt svårt att förstå vad allt gör i början, jag jämförde allting med visual studio och letade efter likheter imellan dom två. Tyckte det var svårt att förstå hur det går till när man matar in ett värde via tangentbordet och skulle få in det i sin kod men efter ett antal timmars testande så fick jag det att fungera. Tror jag kommer bli säkrare på JavaScript om jag håller på med det mer.

Gick det smärtfritt att koppla ihop JavaScript, Node.js med MySQL?
Det var enkelt att förstå hur man öppnar upp filen via konsolen med Node och förstå hur det gick ihop men att använda sql i JavaScript koden var lite klurigare. Men sen när man fick det att fungera så var det ju exakt samma kodning där som man skulle gjorde i sql workbench så då kändes det bättre. Men det var lite klurigt i början som sagt.

Vilken är din TIL för detta kmom?
Definitivt mer om sql den här gången, jag blev lite mer säker på javascript men kan nog inte säga än att jag är riktigt säker på det, krävs lite mer övning innan jag kan säga att jag har lärt mig JavaScript.

Kmom03

Några generella reflektioner på att skriva och jobba med terminalprogram i JavaScript och Node.js?
Har börjar förstå mer nu om JavaScript och kunna jobba med det lite bättre men tycker fortfarande att det är svårt, speciellt i början när man ska komma igång med själva "main" funktionen. Vet inte riktigt var man ska börja eller vilka grejer man ska ha med osv men förstår defenitivt mer nu efter den här veckans uppgifter. Annars tycker jag att det är väldigt smiditgt att man kan föra ihop javascript och Node och SQL.

Lyckas du få till bra kodstruktur i ditt terminalprogram mot databasen?
Jag skrev så som jag brukade skriva i visual studio men fick mycket felmeddelanden i temrminalen när jag validatade min fil men för mig så gör det inte så mycket, tycker det ser snyggt ut efter jag följer hur konsollen vill att det ska se ut så jag klagar inte.

Hur gick det att jobba med ER-modelleringen, några reflektioner?
Tyckte det var lätt att jobba med draw.io och kunna tänka sig hur ens diagram ska vara uppbyggt men själva uppgiften var lite luddig och jag hade svårt att komma igång med den för jag förstod inte riktigt exakt vad jag skulle göra eller hur jag skulle börja. Men jag tror jag gjorde rätt tillslut och det var intressant och roligt att tänkta på det sättet när man skapar något sådant, känns som det är nyttigt till framtiden.

Fanns det några extra svårigheter du kämpade med i kursmomentet?
Jag kände den här veckan att det blev mycket svårare att klara allting själv utan att fastna så känner överlag att den här veckan var mycket tuffare. Fastnade på outer join när jag först kom dit, förstod konceptet hur det skulle fungera men fick det inte riktigt att fungera ändå i början så där fastnade jag länge innan jag kom vidare. Sedan tyckte jag almänt att i den här veckans uppgifter så var det inte så mycket som fungerade enligt instruktionerna, den bash commando filen som jag gjorde fungerade inte alls sen var det flera andra saker som inte heller fungerade när jag följde anvisningarna.

Vilken är din TIL för detta kmom?
Har defenitivt lärt mig en del om javascript den här veckan men kan fortfarande inte riktigt säga att jag är riktigt säker på det än. Jag kände tyvärr inte riktigt att jag har blivit så mycket säkrare på SQL just den här veckan eftersom det var så mycket strul med dom uppgifterna som gjorde att det blev lite frustrerande att fortsätta.
